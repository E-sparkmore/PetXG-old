## 小光桌宠

- 按住左键以移动位置
- 按住中键以缩放大小
- 单击右键以打开菜单

由于QMediaPlayer控件打包后无法链接文件，故打包后没有音乐功能。音乐功能可下载源码在python运行环境下使用。

---

## Desktop Pet

- Hold down the left button to move the position
- Hold down the middle button to zoom in or out
- Right-click to open the menu

Due to the inability to link files after packaging the QMediaPlayer control, there is no music functionality after packaging. The music function can be downloaded and used in a Python runtime environment with the source code.
